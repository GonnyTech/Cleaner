# Release Notes - Windows Master Utility v1.0

**Data di rilascio:** 8 Febbraio 2026

## 🎉 Prima Release Ufficiale

Questa è la prima release stabile di **Windows Master Utility**, l'applicazione completa per pulizia e ottimizzazione Windows.

---

## ✨ Funzionalità Principali

### 🧹 Pulizia Sistema
- Svuota Cestino
- Cancella Download obsoleti (> 7 giorni)
- Pulisci File Temporanei (User + System)
- Pulisci Prefetch
- Cancella Windows.old

### 🗑️ Rimuovi Bloatware
- Rimozione di 14+ app Windows preinstallate
- Dialogo di conferma con elenco app
- Include: Edge, Copilot, Recall, Mail, Camera, Paint 3D, ecc.

### ⚙️ Ottimizzazioni
- Disabilita pubblicità Windows
- Pianificazione automatica settimanale (Domenica 10:00)
- Installazione automatica VS Code

### 📊 Interfaccia Utente
- Tema dark professionale
- Header blu gradient
- 4 schede organizzate
- Log in tempo reale con icone
- Dialogo modale di conferma per rimozioni

---

## 📦 Contenuto Release

- `WindowsMaster.exe` (14.3 MB) - Eseguibile standalone
- `README.md` - Documentazione completa

---

## 🔧 Requisiti di Sistema

- **OS:** Windows 10 o Windows 11
- **Privilegi:** Amministratore (raccomandato)
- **Spazio:** 15 MB

---

## 🚀 Installazione

1. Scarica `WindowsMaster.exe`
2. Esegui come Amministratore
3. Conferma UAC
4. Inizia a pulire!

---

## ⚠️ Note Importanti

- **Backup consigliato:** Crea un punto di ripristino prima di usare il tool
- **Rimozione app irreversibile:** Le app rimosse non possono essere reinstallate facilmente
- **Admin richiesto:** Molte funzioni richiedono privilegi amministratore
